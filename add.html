Contribute to the job board

You can add a job or create a seeker profile. For jobs, we recommend Quick add if the posting is on a supported site.

Recommended

Quick add (paste job link)

Click the button below — it opens a job-url.txt file editor in your fork. Paste the job posting URL as the file contents, commit it to a new branch, then open the pull request. Our bot will scrape the page and add the Markdown file automatically.

Supported sites

  • Greenhouse (e.g. jobs.greenhouse.io, boards.greenhouse.io)
  • Lever (e.g. jobs.lever.co)
  • Workable
  • Other common ATS and career pages (we’ll try to extract title, company, description)
Open a new pull request (quick add)

Manual add

Create a new Markdown file in the jobs/ folder and open a pull request. Use the filename format: company-slug-job-title-slug.md (e.g. acme-senior-engineer.md).

  1. Fork the repository (if you haven’t already).
  2. In your fork, create a new file under jobs/ with the name company-slug-job-title-slug.md.
  3. Use the YAML frontmatter and body format shown in the sample job.
  4. Commit, push, and open a pull request to the main repo.

Create a job seeker profile

If you're looking for opportunities, you can create a public profile that appears on the Job seekers page.

  1. Fork the repository (if you haven’t already).
  2. Create a new file under seekers/ named after you, e.g. jane-doe.md.
  3. Use the YAML frontmatter and body format shown in the sample seeker profile.
  4. Commit, push, and open a pull request to the main repo.
