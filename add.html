Add a job

You can add a listing in two ways. We recommend Quick add if the job is on a supported site.

Recommended

Quick add (paste job link)

Paste the URL of the job posting. We’ll create the listing for you. Just open a pull request and put the job URL in the description (or in a file named job-url.txt); our bot will scrape the page and add the Markdown file.

Supported sites

  • Greenhouse (e.g. jobs.greenhouse.io, boards.greenhouse.io)
  • Lever (e.g. jobs.lever.co)
  • Workable
  • Other common ATS and career pages (we’ll try to extract title, company, description)
Open a new pull request (quick add)

Manual add

Create a new Markdown file in the _jobs/ folder and open a pull request. Use the filename format: company-slug-job-title-slug.md (e.g. acme-senior-engineer.md).

  1. Fork the repository (if you haven’t already).
  2. In your fork, create a new file under _jobs/ with the name company-slug-job-title-slug.md.
  3. Use the YAML frontmatter and body format shown in the sample job.
  4. Commit, push, and open a pull request to the main repo.
